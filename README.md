## Deployment

![Border](images/border.png)

Clone the project

```bash
  git clone https://github.com/DVS-Tech-Open-Source-Projects/DVS-Tech-Moderation-Bot.git
```

Go to the project directory

```bash
  cd DVS-Tech-Moderation-Bot
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```
